FREELANCEFLOW PROJECT IMAGES
=============================

Add 3 screenshots of the FreelanceFlow dashboard to this folder:

1. img1.jpg - Primary screenshot (main dashboard)
2. img2.jpg - Feature showcase (client management or income tracking)
3. img3.jpg - Secondary view (analytics or invoice view)

Requirements:
- JPG format (or PNG, but rename to .jpg)
- Recommended size: 800x600px or larger
- Aspect ratio: 4:3 or 16:9
- File names must match exactly: img1.jpg, img2.jpg, img3.jpg

The carousel will automatically rotate through these images
every 3 seconds with a smooth fade transition.
