LOGO IMAGE
==========

Place your logo image here:

Location: assets/images/logo/logo.png

RECOMMENDED SPECS:
- Format: PNG (transparent background)
- Dimensions: 200x60px or similar ratio
- Style: Simple, readable at small sizes
- Fallback: If logo.png is missing, "AG." text will display

The logo will appear in the navbar with:
- Max height: 40px (32px on mobile)
- Hover effect: slight scale (1.05) and opacity change
- Responsive sizing

To create a simple logo:
1. Use Canva, Figma, or any design tool
2. Create text "AG" or your full name
3. Export as PNG with transparent background
4. Save as logo.png in this folder
